# D3

## Data binding
Conectar los datos a objetos.
```typescript
d3.csv("datos.csv").then(datos=>
{
    const texto2=mySvg.selectAll("text").data(datos)
    texto2.enter().append("text")
    .attr("x",(d,i)=>(i*50)+50)
    .attr("y",50)
    .attr("font-size",14)
    .attr("fill","purple")
    .text((d)=>d.nombre)
})
```

> The function is a promise that inputs the extracted data as a parameter.
> In the formula there are three parts:
> - Part 1: selectAll elements (still not existing) and bind to data
> - Part 2: use enter().append to generate every object
> - Part 3: Use chaining with anonymous functions to introduce attributes in the objects

## Scales
Crear funciones de conversi√≥n de escalas.

### Lineal

- Definir
    ```typescript
    const y=d3.scaleLinear()
        .domain([0,100])
        .range([0,10])
    ```
- Query
    - directa: console.log(y(50)) //5
    - inversa: console.log(y.invert(5)) //10

### Logarithmic

```typescript
    const x=d3.scaleLogarithmic()
        .domain([0,100])
        .range([0,10])
        .base(10)
    console.log(x(10)) //1
    console.log(x.invert(2)) //100
```
### Time scales

```typescript
    const x=d3.scaleTime()
        .domain([new Date(2020,1,1),new Date(2020,12,31)])
        .range([0,100])
    console.log(x(new Date(2020,6,30)) //50
    console.log(x.invert(50)) //Jun 30, 2020
```
