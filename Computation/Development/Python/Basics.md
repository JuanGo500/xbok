# Python basics

## Variables
- A variable is created the moment you first assign a value to it.```x=1```
- Variables do not need to be declared with any particular type, and can even change type after they have been set.
- A variable name cannot start with a number.
- It is possible to assign multiple values ```x, y = "Orange", "Banana"```
- A constant has all letters in uppercase

## Types
- Text: str
- Numeric: int, float, complex
- Structures: tuple, list, dict, range, set, frozenset
- Other: bool, byte, bytearray, memoryview, NoneType
- get the data type ```type(object)```
