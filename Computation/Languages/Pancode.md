# Pancode

## Variables
- Flutter: int myVariable = 5;
- Python: my_variable = 5
- Typescript: let myVariable: number = 5;

## Operators
+ concatenates strings FL-PY-TY

## Data structures
### Lists
- Flutter: my_list = [1, 2, 3, 4, 5];
- Python: my_list = [1, 2, 3, 4, 5]
- Typescript: my_list: number[] = [1, 2, 3, 4, 5]

### Dictionaries
Flutter
```
Map myMap = {
    "Name": "John Doe",
    "Age": 25
};
```

Python
```
mydict = {
    "Name": "John Doe",
    "Age": 25
}
```

Typescript
```
let myDict = {
    "Name": "John Doe",
    "Age": 25
};
```

## Flow
Flutter
```
for(int i = 0; i < 10; i++) { 
  print(i); 
}
```

Python
```
for i in range(10):
    print(i)

for i in range(0, 10, 2):
    print(i)
```

Typescript
```
for(let i = 0; i < 10; i++) {
   console.log(i); 
}
```
