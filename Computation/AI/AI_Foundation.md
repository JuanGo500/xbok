### Machine Learning
## Definition
- Allows computers to use data to predict the future
- The model is trained using *training data*
- The model is evaluated using *testing data*
- A model can be used to:
  - Predict future values
  - Categorize values
  - Recognize objects
  - Understand human language

## Types
- Regression
- Classification
- Clustering
