# D3

```html
<html>
    <body>
        <script src="https:/d3js.org/d3.v7.min.js"> 
        </script>

        <script>
            svg=d3.select("body").append("svg")
                    .attr("width",300)
                    .attr("height",300)
                
            svg.selectAll("circle")
                .data([2,3,4])
                .enter()
                .append("circle")
                    .attr("fill","red")
                    .attr("r",10)
                    .attr("cx",i=>i*20)
                    .attr("cy",i=>i*10);
        </script>

    </body>
</html>
```
- Primero se selecciona el un objeto del DOM
- Luego se asocian los datos
- Mediante enter, se crea un elemento temporal por cada dato si no hay elementos en el dom
- Para cada elemento temporal se crea un append
