# Git

## Key concepts
- Commit: snapshot
- Branch: series of consecutives commits
- Areas: where files are stored
    - Workspace: where untracked and tracked modified files are stored
    - Index: where tracked staged files are stored
    - Local repository: where commited files are stored
    - Remote repository: where commited files are syncronyzed
