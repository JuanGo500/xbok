# SVG
SVG is the syntax to create vectorials graphics in HTML.

# Creating shapes
Shapes are wrapped by svg objects.
## Rectangle
```html
<svg height="100" width="200">
    <rect height="50" width="100" fill="red"></rect>
</svg>
```

## Circle
```html
<svg x="30" y="100" height="100" width="200">
    <circle cx="50" cy="100" r="40" fill="blue"></rect>
</svg>
